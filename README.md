# Data_Analysis_7day

7일 동안 진행한 데이터 분석 학습 및 실습 내용을 정리한 저장소입니다.  
Jupyter Notebook을 중심으로 데이터프레임 처리, 시리즈 다루기, CSV/TXT 파일 입출력, 데이터 시각화 등의 기초 내용을 학습하고 실습한 결과를 포함하고 있습니다.

## 프로젝트 개요

이 저장소는 데이터 분석 기초를 학습하는 과정에서 작성한 노트북과 실습 파일을 모아둔 프로젝트입니다.  
주요 학습 내용은 다음과 같습니다.

- Pandas를 활용한 Series 및 DataFrame 기초
- CSV / TXT 파일 불러오기 및 저장
- 데이터프레임 구조 확인과 기본 전처리
- Matplotlib 기반 데이터 시각화
- 샘플 데이터를 활용한 반복 실습
